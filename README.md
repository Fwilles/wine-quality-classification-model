# **Wine quality classification model**
In this repository I share my recent project in which I created a model for classifying wine quality between bad, good and very good.

-----------------------------------------------------------------------------------

During this project, I had the opportunity to apply various knowledge acquired in the course "Introduction to Machine Learning," offered by Instituto Eldorado, among which I highlight:

- Data treatment and cleaning;
- Identification and management of outliers;
- Data preprocessing;
- Application of machine learning algorithms;
- Model evaluation through metrics;
- Hyperparameter tuning and optimization.

------------------------------------------------------------------------------------

### **Dataset Explanation**
The chosen dataset provides quality data on red wines, including features such as volatility, acidity, sugar, density, pH, quality, and more. The wine quality is rated on a scale from 0 to 10.

### **Business Problem**
My objective with this dataset is to classify each wine as ruim, bom, or muito bom, in order to predict when a wine production will be well received by the consumer market.

------------------------------------------------------------------------------------

In this project, I chose to use the **Random Forest** machine learning algorithm, which is a supervised learning algorithm that creates multiple decision trees that are merged together for more accurate prediction. The "forest" it builds is an ensemble of decision trees, usually trained using the "bagging" method. The general idea behind the bagging method is that a combination of learning models increases the overall result. The logic behind the Random Forest model is that multiple uncorrelated models (the individual decision trees) perform much better as a group than they do on their own.
